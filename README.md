# No-Loss Auction Protocol (Soroban)

## Project Description

This project is a decentralized no-loss auction system built on Stellar using Soroban smart contracts. Users can place bids, track the highest bidder, and finalize auctions after the deadline.

---

## Smart Contract

- Deployed on Stellar Testnet
- Contract ID:
  CCQPT6X5KAMUO6P4DBBWOSQTDNQZ54WV27JQTWTAZI3OFGJHS5CDPEIT

---

## Features

- Create auction
- Place bids
- Track highest bidder
- Finalize auction
- Claim refunds

---

## Frontend

The frontend is built using:

- HTML
- CSS
- JavaScript
- Freighter Wallet integration

It connects directly to the Soroban smart contract deployed on testnet.

## Challenges Encountered

- Integrating Freighter wallet with frontend for transaction signing
- Handling Soroban RPC transaction structure in browser environment
- Fixing network mismatches between testnet and wallet configuration
- Resolving Freighter injection issues in local development
- Understanding contract invocation flow and transaction signing lifecycle
